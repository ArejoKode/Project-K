# Week5 作業 - Advent of Code 2025

這週的作業內容不准用 AI，全部都要自己寫出來，作業有任何問題都可以問我

## 什麼是 Advent of Code？

[Advent of Code](https://adventofcode.com/) (AOC) 是一個每年 12 月舉辦的程式解題挑戰活動，從 12/1 到 12/12 每天會釋出一道新題目。題目難度會隨著天數逐漸增加，是很好的演算法和邏輯思維訓練。

---

## 作業規則

1. **禁止使用 AI** 輔助解題（包括 ChatGPT、Claude、Copilot 等）
2. **限定使用 JavaScript** 撰寫程式碼
3. 每題都有 PART 1 和 PART 2, 第一天的 PART1 和 PART2 都要解，第二天只要解 PART1 就好 (如果你想挑戰自己，可以把第二天的 PART2 也解掉)
4. Day3 是挑戰題，一樣有時間再做就好
